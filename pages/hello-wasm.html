Hello WASM - Student Template

About This Template

This is a simplified template designed for students to learn from. It demonstrates the core WASM state management pattern used throughout this project.

Key Learning Points:

  • State Management: Uses LazyLock<Mutex<State>> pattern in Rust
  • WASM Bindings: Shows how to export functions with #[wasm_bindgen]
  • TypeScript Integration: Demonstrates loading and using WASM modules
  • Route Handler Pattern: Shows how to structure route initialization

To Extend This Template:

  1. Add new fields to HelloState in wasm-hello/src/lib.rs
  2. Export new functions with #[wasm_bindgen]
  3. Add function signatures to wasmModuleExports in src/routes/hello-wasm.ts
  4. Add UI elements here and event handlers in the route handler
